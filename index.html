Change Log:

    Added:

  • Model Mirroring.
    • You can now easily mirror any part or shape on the x or z axis with or without mirroring the textures . (note : mirroring on the x axis is somewhat broken with cubes due to Mine-Imator not having the ability to mirror cube textures on the x axis . )
  • Orthographic Projection / Isometric Projection
    • You can also view your models in Orthographic View, this is might become useful to you when taking model screenshots.
  • View Snapping
    • You can now snap the camera view to the X, Y or Z axis to make aligning objects when modeling easier.
  • Home screen Thumbnails
    • Easily view the models you open from the Home page.
  • Key-bind Transforms
    • Easily move selected elements with your keyboard instead of using gizmos . (note : Free rotation is not yet available and you have to press the X, Y or Z buttons when using the rotation key-bind)
  • View Element
    • Easily look at the current selected element.
  • Add Key-bind
    • Quickly add Planes, 3D-Planes, Cubes or Parts using a key-bind.
  • Compact Elements and Panels
    Reduce the spacing between UI elements like how it is in Mine-Imator.
  • FOV Slider
    • Control the FOV of the work camera to your desirable value.
  • Draggable joint offset
  • Back to home screen button
  • Optional duplicate name warnings
    • Parts with the same name will show an error icon.

    Bug-fixes :

  • Memory Leak Bug
    • An issue existed with Modelbench 1.1.5 where the program used to use excessive amounts of ram this issue is now fixed.
  • Open Model has the New Model Keybind in the file context menu.
  • Center Pivot Axis not working correctly.
  • Reset Pivot offset not working correctly.
  • Pasting Pivot Leads to Modelbench crashing.
  • Freeze when exporting an obj with zero shapes.
  • Ability to adjust elements in preview mode.
